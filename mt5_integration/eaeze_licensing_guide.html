EAEZE Systems: MQL5 License & Sci-Fi Sync Integration Guide

คู่มือการใช้งานระบบลิขสิทธิ์และการซิงก์ข้อมูลพอร์ตเทรดเรียลไทม์ขึ้นแดชบอร์ด (Simple Sci-Fi Dashboard) ผ่านไฟล์ห้องสมุดส่วนกลาง เพื่อความมั่นคงปลอดภัยและความรวดเร็ว

1. การเตรียมไฟล์สำหรับนักพัฒนา EA (Setup Files)

นำไฟล์ต่อไปนี้ไปวางติดตั้งใน MetaEditor ของท่าน:

1. EAE_Licensing_Lib.ex5 (ไฟล์ห้องสมุดไลบรารีที่คอมไพล์แล้ว)

  • 📂 นำไปวางในโฟลเดอร์: MQL5/Libraries/

    • 2. EAE_Licensing.mqh (ไฟล์ส่วนหัวสำหรับใช้เรียกนำเข้าฟังก์ชัน)

  • 📂 นำไปวางในโฟลเดอร์: MQL5/Include/
    • IMPORTANT: ห้ามนำไฟล์ซอร์สโค้ดต้นฉบับ .mq5 ของไลบรารีไปแจกจ่ายให้ผู้อื่นเด็ดขาดเพื่อความปลอดภัยของคีย์และฐานข้อมูลหลังบ้าน

    2. ขั้นตอนการติดตั้งเข้าสู่ตัว EA (Integration Steps)

    ให้นักพัฒนาเปิดโค้ด EA ที่ต้องการเช็คสิทธิ์ใน MetaEditor และเขียนโค้ดเพิ่ม 4 ขั้นตอนสั้นๆ ดังต่อไปนี้:

    ขั้นตอนที่ 1: แนบไฟล์นำเข้าห้องสมุด (Top of code)

    // 1. นำเข้าไฟล์ไลบรารีส่วนกลาง EAE_
#include <EAE_Licensing.mqh>

// กำหนดรหัสสินค้า EA ของท่านในระบบเว็บ
#define EA_PRODUCT_ID "EZM-MIN-V1"

    ขั้นตอนที่ 2: เพิ่มการตรวจสอบตอนเริ่มทำงาน (OnInit)

    int OnInit()
{
   // 2. เรียกใช้งานระบบเช็คสิทธิ์ช่วงเริ่มทำงาน
   if(!EaezeCheckLicense(EA_PRODUCT_ID))
   {
      Print("EAE_SYSTEM: License Check Failed. Stopping EA.");
      return(INIT_FAILED); // ปิดตัวลงทันทีหากสิทธิ์ไม่ถูกต้อง
   }
   
   EaezeRemoveLicenseAlert();
   
   // ... เขียนโค้ดเริ่มต้นอื่นๆ ของท่านต่อจากนี้ ...
   return(INIT_SUCCEEDED);
}

    ขั้นตอนที่ 3: เพิ่มตัวเช็คสิทธิ์และซิงก์ข้อมูลระหว่างทำงาน (OnTick / OnTimer)

    void OnTick()
{
   // 3. ตรวจสอบสิทธิ์และส่งข้อมูลขึ้นแดชบอร์ดไซไฟอัตโนมัติ (Smart Sleep/Wake Sync)
   // พารามิเตอร์: Product ID, System Code, EA Version, Sync Interval (วินาที), Magic Buy (0=สแกนทั้งหมด), Magic Sell (0=สแกนทั้งหมด)
   EaezeCheckLicenseAndSync(EA_PRODUCT_ID, "EASYM", "1.12", 20, 0, 0);
   
   // ... เขียนโค้ดเทรดปกติของ EA ของท่านต่อจากนี้ ...
}

    ขั้นตอนที่ 4: ล้างป้ายเตือนความปลอดภัยเวลากดลบ EA (OnDeinit)

    void OnDeinit(const int reason)
{
   // 4. ล้างกล่องข้อความเตือนความปลอดภัยเมื่อหยุดใช้งาน
   if(reason != REASON_INITFAILED)
   {
      EaezeRemoveLicenseAlert();
   }
}

    3. ความปลอดภัยของระบบตัวแปรเครื่อง (F3 Protection)

    ระบบไลบรารีส่วนกลางนี้จะเก็บค่าผลการเช็คสิทธิ์ไว้ในตัวแปรพอร์ตเครื่องพร้อมแนบลายเซ็นดิจิทัลเฉพาะ (Digital Checksum) หากผู้ใช้พยายามแก้ไขสเตตัสใน Global Variables ด้วยคีย์ F3 ระบบความปลอดภัยของไลบรารีจะรับรู้การเปลี่ยนแบบไม่ถูกต้องและสั่งถอดถอน EA ออกจากกราฟทันทีเพื่อความปลอดภัยสูงสุด
