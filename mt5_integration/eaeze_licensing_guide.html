EAEZE Systems: MQL5 License Verification Library Integration Guide

คู่มือการใช้งานระบบลิขสิทธิ์เทรดเรียลไทม์ผ่านไฟล์ห้องสมุด (Library) เพื่อป้องกันความปลอดภัยของซอร์สโค้ดและข้อมูลระบบหลังบ้าน 100%

  • --

    • 1. การเตรียมไฟล์สำหรับนักพัฒนา EA (Setup Files)

    ทาง EAEZE จะส่งไฟล์ให้นักพัฒนาภายนอกนำไปติดตั้งในเครื่อง MetaEditor 2 ไฟล์ดังนี้:

    1. EAEZE_Licensing_Lib.ex5 (ไฟล์ห้องสมุดที่คอมไพล์แล้ว)

  • 📂 นำไปวางในโฟลเดอร์: MQL5/Libraries/

    • 2. EAEZE_Licensing_Imports.mqh (ไฟล์นำเข้าฟังก์ชัน)

  • 📂 นำไปวางในโฟลเดอร์: MQL5/Include/
    • IMPORTANT: ห้ามแจกจ่ายซอร์สโค้ด .mq5 ของไลบรารีให้ผู้อื่นเด็ดขาด เพื่อป้องกันคีย์ฐานข้อมูลและ URL หลังบ้านรั่วไหล
  • --

    • 2. ขั้นตอนการติดตั้งเข้าสู่ตัว EA (Integration Steps)

    ให้นักพัฒนาเปิดโค้ด EA ที่ต้องการเช็คสิทธิ์ใน MetaEditor และเขียนโค้ดเพิ่ม 4 ขั้นตอนสั้นๆ ดังต่อไปนี้:

    ขั้นตอนที่ 1: แนบไฟล์นำเข้าห้องสมุด (Top of code)

    ให้เขียนโค้ดนี้ไว้ที่บรรทัดบนสุดของไฟล์ EA ของท่าน:

    // 1. นำเข้าไลบรารีเช็คลิขสิทธิ์ EAEZE

    #include <EAEZE_Licensing_Imports.mqh>
    


    // กำหนดรหัสสินค้า EA ของท่านในระบบเว็บ (Product Key เช่น EZM-MAX-V1)
    

    #define EA_PRODUCT_ID "EZM-MAX-V1"

  • --

    • ขั้นตอนที่ 2: เพิ่มการตรวจสอบตอนเริ่มทำงาน (OnInit)

    ให้เขียนบล็อกโค้ดนี้ไว้ในฟังก์ชัน OnInit() ก่อนเริ่มรันระบบคำนวณอื่นๆ:

    int OnInit()

    {
    

       // 2. เรียกใช้งานระบบเช็คสิทธิ์ทันทีตอนติดตั้ง EA
    

       if(!EaezeCheckLicense(EA_PRODUCT_ID))
    

       {
    

          Print("EAEZE: License Check Failed. Stopping EA.");
    

          return(INIT_FAILED); // สิทธิ์ใช้งานไม่ถูกต้อง บังคับหยุดการทำงาน
    

       }
    

       
    

       // ... เขียนโค้ดของท่านต่อจากบรรทัดนี้ ...
    

       return(INIT_SUCCEEDED);
    

    }

  • --

    • ขั้นตอนที่ 3: เพิ่มตัวเช็คสิทธิ์ระหว่างทำงาน (OnTick / OnTimer)

    ให้ใส่ฟังก์ชันนี้ไว้ที่ บรรทัดแรก ของฟังก์ชัน OnTick() เพื่อให้ระบบเช็คสิทธิ์ตามรอบอย่างสม่ำเสมอ (เช็คเรียลไทม์ผ่านระบบจำในเครื่อง ไม่ทำให้บอทสะดุด):

    void OnTick()

    {
    

       // 3. ตรวจสอบสิทธิ์ระหว่างรันเทรด (ระบบจะดีดบอทออกทันทีถ้าหมดอายุหรือทุนไม่ถึง)
    

       EaezeCheckLicensePeriodic(EA_PRODUCT_ID);
    

       
    

       // ... เขียนโค้ดของท่านต่อจากบรรทัดนี้ ...
    

    }

    *หมายเหตุ: หาก EA ของท่านทำงานเป็นรอบเวลา สามารถเอาไปใส่ในฟังก์ชัน OnTimer() แทนได้เช่นกัน*

  • --

    • ขั้นตอนที่ 4: ล้างป้ายเตือนความปลอดภัยเมื่อหยุดใช้งาน (OnDeinit)

    ให้เขียนโค้ดนี้ไว้ในฟังก์ชัน OnDeinit() เพื่อลบกล่องแจ้งเตือนสีแดงเวลากดลบ EA ออกจากกราฟ:

    void OnDeinit(const int reason)

    {
    

       // 4. ลบป้ายแจ้งเตือนออกจากกราฟกรณีปิดใช้งานหรือปิดกราฟ
    

       if(reason != REASON_INITFAILED)
    

       {
    

          EaezeRemoveLicenseAlert();
    

       }
    

       
    

       // ... เขียนโค้ดของท่านต่อจากบรรทัดนี้ ...
    

    }

  • --

    • 3. คู่มือการตั้งค่าสำหรับลูกค้าผู้ใช้งาน (User Settings)

    เพื่อให้ระบบเช็คสิทธิ์ทำงานได้ ลูกค้าที่นำ EA ไปติดตั้งจะต้องตั้งค่าในโปรแกรม MT5 ดังนี้:

    1. ไปที่เมนู Tools > Options (หรือกดปุ่ม Ctrl + O)

    2. เลือกแท็บ Expert Advisors

    3. ติ๊กเครื่องหมายถูกที่ช่อง [✓] Allow WebRequest for listed URL

    4. กดปุ่ม Add new URL (เครื่องหมายบวก) แล้วพิมพ์ URL ต่อไปนี้ลงไป:

  • https://eaeze.com

5. กดปุ่ม OK
